Configuration Validator

Ensure your YAML files are correct before deploying them to the server.

YAMLValidator

Detect syntax errors, missing keys, and malformed structures with precise line and column reporting.

validate_file(path) / async validate_file_async(path)

Validate a local YAML file and get a detailed result object.

validate_string(content)

Validate a YAML string directly.

JSON Export Example

from mcnexus import YAMLValidator

result = YAMLValidator.validate_string("key: : value") # Broken YAML
print(result.to_json())

Validation Models

The YAMLValidationResult provides a structured overview of the check.

to_dict() / to_json()

Convert the validation result into a dictionary or a JSON string for API integrations.

FieldDescription
is_valid Boolean indicating if the YAML is syntactically correct.
error A YAMLValidationError object if is_valid is False.
