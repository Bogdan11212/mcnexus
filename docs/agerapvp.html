AgeraPVP API

Complete asynchronous implementation of the AgeraPVP Public API. Access everything from player stats to global leaderboards and staff monitoring.

Player API

async get_profile(name) / async get_stats(name, mode)

Fetch general profile data or specific mode statistics (BW, DUELS, SKYWARS, etc.).

async get_friends(name) -> List[Dict]

Retrieve the full list of a player's friends profiles.

Server & Core Monitoring

async get_total_online() / async get_uptime()

Monitor the project's health and total player count.

async get_leaderboard(mode, field) -> Dict[str, int]

Generate a leaderboard for any mode and field (e.g., BW, wins). Note: Takes 1-5s to generate.

async get_streams() -> List[Dict]

Get a live list of running streams from AgeraPVP content creators.

Staff & Security

async get_staff_stats() / async get_online_staff()

Access moderation statistics (bans/mutes) and see which staff members are currently online.

Comprehensive Example

from mcnexus import AgeraPVPAPI
import asyncio

async def main():
    async with AgeraPVPAPI(api_key="YOUR_KEY") as agera:
        # Check overall online
        online = await agera.get_total_online()
        print(f"Total Players: {online}")

        # Get top 10 BedWars winners
        top = await agera.get_leaderboard("BW", "wins")
        print(f"Top Winners: {top}")

        # Check online staff
        staff = await agera.get_online_staff()
        print(f"Online Moderators: {[s['displayName'] for s in staff]}")

asyncio.run(main())
