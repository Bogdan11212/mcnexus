AgeraPVP API

Full integration with the AgeraPVP server API. Track player stats, profiles, and render high-quality visuals using their dedicated skin service.

AgeraPVPAPI

async get_profile(name_or_uuid) -> Dict

Fetch general profile information including rank and group.

async get_stats(name_or_uuid) -> Dict

Access detailed game statistics (BedWars, Duels, etc.).

API Usage Example

from mcnexus import AgeraPVPAPI

async def show_stats():
    # API Key is optional for some public endpoints
    async with AgeraPVPAPI() as agera:
        profile = await agera.get_profile("PlayerName")
        stats = await agera.get_stats("PlayerName")
        print(f"Stats: {stats}")

import asyncio
asyncio.run(show_stats())
