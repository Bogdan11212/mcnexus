Log Engine

Turn static text files into live event streams. No Java plugins required.

LogWatcher

on_event(event_type, callback)

Registers an async handler for events like ChatEvent, DeathEvent, etc.

from mcnexus import LogWatcher, ChatEvent

async def my_handler(event: ChatEvent):
    print(f"[{event.player}] {event.message}")

watcher = LogWatcher("logs/latest.log")
watcher.on_event(ChatEvent, my_handler)
await watcher.start()
