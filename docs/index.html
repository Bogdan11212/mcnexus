Introduction

mcnexus is an asynchronous, high-performance Python library for Minecraft server management. It provides low-level protocol implementations wrapped in a developer-friendly API.

Installation

pip install git+https://github.com/Bogdan11212/mcnexus.git
Class

RCONClient

The primary class for executing commands on a single server. Features automatic synchronization and re-connection.

Methods

async command(cmd: str) -> RCONResponse

Sends a command to the server. Handles multi-packet responses automatically.

async connect()

Establishes connection and authenticates. Not needed if using async with.

async disconnect()

Safely closes the connection.

Example

from mcnexus import RCONClient

async with RCONClient("127.0.0.1", 25575, "password") as client:
    response = await client.command("list")

Class

RCONPool

Manages multiple server connections. Perfect for network-wide announcements or status synchronization.

add_server(alias, host, port, password, ...)

Adds a server to the pool with a unique alias.

async broadcast(cmd: str) -> Dict[str, RCONResponse]

Sends command to ALL servers in parallel.

Class

RCONWatcher

Simulates an event system by polling commands at fixed intervals.

watcher = RCONWatcher(client, interval=1.0)

# Triggers when 'joined' is found in 'list' output
watcher.on_output_contains("list", "joined", my_callback)
await watcher.start()
Function

status()

The swiss-army knife for server information. It follows this logic:

  • Resolves SRV records automatically (handles mc.hypixel.net).
  • Attempts Modern 1.7+ JSON Ping.
  • If fails, attempts Legacy Pre-1.7 Ping.
from mcnexus import status
res = await status("play.example.com", port=25565, timeout=5.0)
Function

status_bulk()

Efficiently pings multiple servers concurrently using asyncio.gather.

results = await status_bulk([("server1.com", 25565), ("server2.com", 25565)])
