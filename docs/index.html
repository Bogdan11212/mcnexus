RCON Module

The RCON module provides a robust implementation of the Source RCON protocol specifically tuned for Minecraft's quirks, such as multi-packet responses.

mcnexus automatically handles packets larger than 4096 bytes using a "dummy packet" synchronization technique.

example_rcon.py 
import asyncio
from mcnexus import RCONClient

async def main():
    # Context manager handles connection & cleanup
    async with RCONClient("127.0.0.1", 25575, "secret") as client:
        response = await client.command("help")
        
        print(f"Server responded in {response.time_taken:.2f}s")
        print(response.clean) # Auto-stripped color codes!

asyncio.run(main())

Server Status (SLP)

Get real-time data from any Minecraft server. Supports SRV record resolution and fallback to Legacy protocols.

check_status.py 
from mcnexus import status

async def check():
    # SRV resolution is automatic! (e.g. hypixel.net -> IP:Port)
    info = await status("mc.hypixel.net")
    
    if info.online:
        print(f"Players: {info.players_online}/{info.players_max}")
        print(f"Ping: {info.ping:.1f}ms")
        print(f"Version: {info.version_name}")

import asyncio
asyncio.run(check())
