Minecraft Servers.

mcnexus is a production-grade Python library designed for developers who need extreme reliability. Whether you are managing a single server or a network of thousands, we provide the tools to make it happen.

Quick Installation

Install the latest version directly from GitHub. mcnexus requires Python 3.8+ and AsyncIO.

bash 
pip install git+https://github.com/Bogdan11212/mcnexus.git

RCON Core

Minecraft's RCON implementation is notoriously buggy, especially with long responses. mcnexus solves this by implementing a unique Synchronization Marker technique.

The RCONClient Class

Parameter Type Description
host str IP address or domain of the server.
port int RCON port (default: 25575).
auto_reconnect bool Automatically fix broken connections.
python 
from mcnexus import RCONClient

async def run_command():
    # Use context manager for absolute safety
    async with RCONClient("127.0.0.1", 25575, "mypass") as rcon:
        # Long command? No problem.
        res = await rcon.command("help")
        print(f"Server says: {res.clean}")

Server Status (SLP)

Retrieve everything about a server without logging in. Our status module is the most compatible on the market, supporting everything from **Classic 2011 servers** to the latest **1.20+ releases**.

Modern SLP

Fast JSON-based protocol for versions 1.7 and above. Supports favicons and player samples.

Legacy SLP

Fallback for older versions. Automatically engaged if the modern ping fails.

SRV Resolution

Full DNS support. It finds the real IP behind records like `mc.hypixel.net` instantly.

Precise Ping

Latency calculation is done at the protocol level for extreme accuracy.

python 
from mcnexus import status

async def probe():
    server = await status("play.myserver.com")
    
    if server.online:
        print(f"[{server.ping:.1f}ms] {server.players_online} players")
        print(f"MOTD: {server.motd_clean}")
