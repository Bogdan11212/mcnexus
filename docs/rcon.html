RCON Protocol

Robust implementation of the Source RCON protocol for Minecraft, with multi-packet response support and auto-reconnect.

RCONClient

async command(cmd: str) -> RCONResponse

Executes a command. Handles large outputs by using a synchronization dummy packet.

from mcnexus import RCONClient

async with RCONClient("127.0.0.1", 25575, "password") as client:
    res = await client.command("help")
    print(res.clean)

RCONPool

async broadcast(cmd: str) -> Dict[str, RCONResponse]

Sends a command to all servers in the pool simultaneously.

RCONWatcher

Simulates events by polling commands at an interval.

on_output_contains(command, substring, callback)

Trigger callback when specific text is detected in the command output.
