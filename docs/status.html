Server Status (SLP)

Retrieve real-time data from any Minecraft server. Automatic SRV resolution and protocol fallbacks included.

Single Server Query

async status(host, port=25565, timeout=5.0) -> StatusResponse

Pings a server and returns a rich object with players, version, and favicon.

from mcnexus import status

res = await status("mc.hypixel.net")
print(f"Latency: {res.ping}ms")

Bulk Scanning

async status_bulk(servers: List[tuple]) -> List[StatusResponse]

Check hundreds of servers concurrently using asyncio.
