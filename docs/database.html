Database Migration

Automate the transfer of data from local SQLite databases (used by most Minecraft plugins) to powerful cloud databases like MySQL, PostgreSQL, and MongoDB. Schema translation is fully automatic!

SQLiteMigrator

async SQLiteMigrator.to_mysql(sqlite_path, mysql_config) -> Dict

Reads the SQLite schema, creates the equivalent tables in MySQL, and moves all data in batches.

async SQLiteMigrator.to_postgres(sqlite_path, pg_config) -> Dict

Translates SQLite schemas to PostgreSQL compatible types (e.g. REAL to DOUBLE PRECISION) and streams the rows.

async SQLiteMigrator.to_mongodb(sqlite_path, mongo_uri, database_name) -> Dict

Transforms relational SQLite tables into MongoDB collections, converting rows into JSON documents dynamically.

Migration Example (MySQL)

import asyncio
from mcnexus import SQLiteMigrator

async def migrate_luckperms():
    # Database config for the target cloud DB
    mysql_config = {
        "host": "db.example.com",
        "port": 3306,
        "user": "minecraft",
        "password": "secretpassword",
        "db": "luckperms_db"
    }
    
    # Run the migration
    stats = await SQLiteMigrator.to_mysql("luckperms.sqlite", mysql_config)
    
    print(f"Migrated {stats['tables']} tables and {stats['rows_migrated']} rows successfully!")

asyncio.run(migrate_luckperms())
