Skins & Renders

High-performance engine for fetching and rendering Minecraft player visuals.

SkinsAPI

Generate high-quality 2D/3D renders using Crafatar and Visage providers.

get_avatar_url(uuid, size=64) / get_body_url(uuid, size=180)

Instant URL generation for various render types.

async download_image(url, save_path)

Asynchronously download any render to your local machine.

from mcnexus import SkinsAPI

async with SkinsAPI() as skins:
    # Get a 3D body render URL
    url = skins.get_body_url("UUID_HERE", size=256)
    
    # Save it locally
    await skins.download_image(url, "player_skin.png")
